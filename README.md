# Nando's Periometer Mood


# Local development

## Server

`npm install`

`npm run dev`

## Client

`cd client`

`npm install`

`npm run serve`

Site available on [localhost:8081](http://localhost:8081)


# Deploy 

Build client public folder

`npm run build-client`


Push changes to `main` branch
